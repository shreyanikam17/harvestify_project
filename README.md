# Harvestify – Interactive Soil Data Dashboard

## Overview

Harvestify is a React-based web application that provides an interactive dashboard to manage and analyze soil data efficiently. 
It enables structured data handling and supports smart agricultural decision-making.

## Features

* Interactive data table
* Search, filter, and sorting functionality
* Add, edit, and delete rows
* Pagination support
* Crop distribution summary
* Export data to CSV

## Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* JSON

## Installation & Setup

npm install
npm start
Open in browser:
http://localhost:3000

## Purpose

The purpose of this project is to build an interactive soil data management system that helps in organizing, visualizing, and exporting agricultural data for better and data-driven farming decisions.
